##### 1. 문자찾기
아스키코드를 이용한 문자 변환
대문자 A~ Z : 65~90
소문자 a~z : 97~122
##### 2. 팰린드롬(replaceAll 정규식 이용)
숫자를 제거 -> [^A-Z]로 영문자가 아니면 ""로 대치
- 이메일 형식 체크
  xxxx@xxxx.xxx 
  String regex = "^[0-9a-zA-Z]+@[0-9a-zA-Z]+(\\.[_0-9a-zA-Z-]+)*$";
-> 여기서 특수문자인 '.'은 \.이렇게 표현하지만 한번 더 감싸준다.  

- 전화번호   
  010 혹은 02로 시작하는 번호
  (02|010)-(02|010)-[\d]{3,4}-[\d]{4}
